# 🍽 Smart Nutrition Recommendation System using AI

## 📌 Project Description
This project is a Smart Nutrition Recommendation System that suggests diet plans based on user inputs like age, weight, height, and fitness goals using Machine Learning.

---

## 🎯 Objectives
- Provide personalized diet recommendations
- Predict daily calorie and protein requirements
- Help users maintain a healthy lifestyle

---

## ⚙️ Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn (Machine Learning)

---

## 🤖 Machine Learning Model
We used **Linear Regression** to predict:
- Calories needed
- Protein requirement

---

## 📊 Features
- User input (Age, Weight, Height, Goal)
- AI-based prediction
- BMI calculation
- Food recommendations
- Health warning system
